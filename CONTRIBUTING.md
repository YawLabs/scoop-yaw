# Contributing

Thanks for your interest in contributing!

## How This Repo Works

This repository contains package manifests that are automatically generated. If you spot an issue with a formula or manifest, please open an issue rather than submitting a PR directly — most changes here are driven by upstream releases.

## Reporting Issues

If a package isn't installing correctly or a version is out of date, open a GitHub issue with:
- The command you ran
- The error output
- Your OS and package manager version

## Pull Requests

All PRs require approval from a maintainer before merging. Direct pushes to the default branch are blocked.

If you do submit a PR:
1. Fork the repo and create a branch
2. Make your change
3. Write a clear description of what you changed and why
